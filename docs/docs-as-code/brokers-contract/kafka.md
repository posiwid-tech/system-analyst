Kafka Basics
Что такое Kafka

Apache Kafka — это распределённый брокер сообщений, предназначенный для обработки потоков данных в реальном времени.

Основные компоненты
Producer — отправляет сообщения
Broker — хранит и передаёт сообщения
Consumer — читает сообщения
Особенности
Высокая пропускная способность
Работа с partition
Хранение offset
Взаимодействие

Producer и Consumer взаимодействуют с Kafka через собственный бинарный протокол поверх TCP.
